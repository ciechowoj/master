# Building

Required dependencies:
* g++
* python-setuptools
* libtbb-dev
* libopenexr-dev
* zlib1g-dev
* OpenEXR (`apt-get install libopenexr-dev`)
* ispc (`ispc.github.io/downloads.html`)

Depenedencies provided as git submodules:
* assimp
* embree
* glad
* glfw
* glm
* googletest
* imgui

To build project run `make` command in main directory.
